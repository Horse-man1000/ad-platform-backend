Context:
- OAuth, Accounts, Campaigns, Ads are complete and stable.
- This phase is READ-ONLY.

--------------------------------
SCHEMA (REQUIRED)
--------------------------------
Add a new Prisma model for daily ad metrics.

Model (conceptually):
- AdMetric
- adId (FK -> Ad)
- date (YYYY-MM-DD)
- impressions (int)
- clicks (int)
- spend (decimal)
- platform (google/meta)

Constraints:
- Unique by (adId + date)
- Idempotent upsert only
- No automatic deletes

Create and apply a Prisma migration.

--------------------------------
ENDPOINT
--------------------------------
Implement:
GET /metrics/sync?clientId=N&from=YYYY-MM-DD&to=YYYY-MM-DD

--------------------------------
BEHAVIOR
--------------------------------
- Validate clientId and date params
- Load client’s ads from DB
- For each connected platform:
  - Refresh token if needed
  - Fetch daily metrics for each ad in range
  - Upsert AdMetric rows idempotently
- Partial platform failures must NOT crash the whole request
- Return a deterministic summary response
